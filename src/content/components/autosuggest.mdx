## Usage

```html
<cs-autosuggest placeholder="Search..."></cs-autosuggest>

<script>
  const autosuggest = document.querySelector('cs-autosuggest');
  autosuggest.options = [
    { value: 'Option 1' },
    { value: 'Option 2' },
    { value: 'Option 3' },
  ];
</script>
```
