## Usage

```html
<cs-box variant="h2">Section title</cs-box>
<cs-box variant="p" color="text-body-secondary">Description text</cs-box>
<cs-box padding="s">Padded content</cs-box>
<cs-box variant="code">inline code</cs-box>
```
