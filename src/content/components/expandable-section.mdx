## Usage

```html
<!-- Collapsed by default -->
<cs-expandable-section header-text="Additional information">
  Hidden content here.
</cs-expandable-section>

<!-- Starts expanded -->
<cs-expandable-section header-text="Details" expanded>
  Visible content here.
</cs-expandable-section>
```
