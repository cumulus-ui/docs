## Usage

```html
<cs-radio-group id="my-radios" value="first"></cs-radio-group>
<script>
  document.getElementById('my-radios').items = [
    { value: 'first', label: 'First option' },
    { value: 'second', label: 'Second option' },
    { value: 'third', label: 'Third option' },
  ];
</script>
```
