## Usage

```html
<!-- Bottom position (default) -->
<cs-split-panel header="Details" position="bottom">
  <p>Panel content displayed below the main content.</p>
</cs-split-panel>

<!-- Side position -->
<cs-split-panel header="Preview" position="side">
  <p>Panel content displayed beside the main content.</p>
</cs-split-panel>
```
