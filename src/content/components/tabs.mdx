<script>{`
  const el = document.getElementById('demo-tabs');
  if (el) {
    const overview = document.createElement('cs-space-between');
    overview.setAttribute('size', 'l');
    overview.setAttribute('direction', 'vertical');
    overview.innerHTML = '<cs-key-value-pairs id="tab-kv"></cs-key-value-pairs>';

    el.tabs = [
      { label: 'Overview', id: 'overview', content: overview },
      { label: 'Monitoring', id: 'monitoring', content: 'CloudWatch metrics and alarms will appear here.' },
      { label: 'Tags', id: 'tags', content: 'Resource tags will appear here.' },
      { label: 'Activity', id: 'activity', disabled: true },
    ];
    el.activeTabId = 'overview';
  }
`}</script>

> **Note:** Tabs are set via the `tabs` JavaScript property. See usage below.

## Usage

```html
<cs-tabs></cs-tabs>

<script>
  const tabs = document.querySelector('cs-tabs');
  tabs.tabs = [
    { label: 'Overview', id: 'overview' },
    { label: 'Details', id: 'details' },
    { label: 'Settings', id: 'settings', disabled: true },
  ];
  tabs.activeTabId = 'overview';
</script>
```
